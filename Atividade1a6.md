# Atividade1-SegundoBimestre-PrimeiroAno
Atividedes

1 - Hello World. Ele executa um comando "escreva" e dentro do campo deste codigo, possui um "Olá Mundo", assim mostrando (ou escrevendo) "Olá Mundo" no Console.

2 - Além do "Olá Mundo" ser mostrado visualmente igual ao código anterior, ele agora vem com uma frase ("Como Vai") a mais que fica colada ao "Olá Mundo"

3 - Semelhante ao código anterior, demonstra as mesmas frases, porém utilizada do "\n" (Quebra de Linha) para descolar (oque acontecia no código anterior), pulando uma linha e escrevendo a segunda frase.

4 - Este código cria uma váriavel string (cadeia) e de um input (leia) para determinar o "valor" da váriavel e após a resposta, ele mostra no console pelo comando "escreva".

5 - Mostrou 123, pulou uma linha, mostrou 300 pela soma dos valores 100+200, pulou uma linha e mostrou "100+200" por estar em cadeia inves de real ou inteiro, a diferença é o tipo de valor, onde um é soma de inteiros e o outro é apenas uma string.

6 - Ele verifica se a idade é maior de 18 anos, se sim, ele mostra a mensagem "maior de idade", como a váriavel idade (21) é maior que 18, ele mostra a mensagem.
